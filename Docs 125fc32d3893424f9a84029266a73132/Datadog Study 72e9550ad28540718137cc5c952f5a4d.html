O que é Datadog

Ferramenta de monitoramento,

  • Cloud,
  • Infra [FOCUS]
  • RUM(Real User Monitoring),
  • Log, [Splunk]
  • APM(aplication Perfomance Monitoring), [NewRelic]
  • Container,
  • Security

Estrutura

Organização geral

Monitoramento como Serviço em Cloud

  • O agente é instalado no Server

Onde conversa com o Serviço do Datadog, no nosso caso Azure.

Terminologia

  • Agent

    O que roda na hosts que queremos monitorar, podendo reportar processos,logs e traces ao vivo.

    É constituído de

    UDP:8125→[Coletor](HTTP) + opcional(HTTP) →TCP:17123→Forwarder → HTTPS→DatadogSaaS

    Faz isso através de Opcionais:

    • DogStatsD

    Roda como Daemon, e envia os dados para o

    • Integrations
    • API
